» Installation Instructions
  1. Unzip yourls1.0.0.zip
  2. Rename includes/config-sample.php to includes/config.php
  3. Open up includes/config.php and fill in the required settings
  4. Upload the unzipped files to your domain public_html or www folder
  5. Point your browser to http://yoursite.com/install.php

» FAQ
Requirements
  1. At least PHP 4.3
  2. At least MYSQL 4.1
  3. PHP CURL extension installed
  4. PHP BCCOMP extension for handling large integers
Limitations
  1. Maximum length of custom keyword is 12 characters
  2. 4,738,381,338,321,617,846 available URLs
Difference Between Base 36 And Vase 62 Encoding
  1. Base 36 encoding uses 0123456789abcdefghijklmnopqrstuvwxyz
  2. Base 62 encoding uses 0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz
